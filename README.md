## ⟳ What is Rotateur?

A minimal desktop app that lets you load any website and rotate it freely — from 0 to 360 degrees.

Built for one reason: **watching videos while lying on your side** without twisting your neck. Load YouTube, Netflix, Twitch, or any streaming site, rotate 90°, and you're done.


---

## ✦ Features

- **Any website** — YouTube, Twitch, Netflix, Cinechill, and everything else. Full Chromium engine, no restrictions.
- **Free rotation** — slider from 0° to 360°, or jump with the quick presets.
- **Sidebar UI** — controls tucked into a 48px side panel so your content gets the full screen.
- **Keyboard shortcuts** — `←` `→` to snap between angles, `Enter` to load.
- **Neo-futuristic interface** — dark, blue-lit, built to look good in a dark room.

---

## ⬇ Getting Started

**Prerequisites:** [Node.js](https://nodejs.org) 18+

```bash
# Clone
git clone https://github.com/0ri0546/Rotator.git
cd Rotator/Rotator

# Install
npm install

# Run
npm start
```

---

## ⌨ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Enter` | Load URL |
| `→` | Rotate +90° |
| `←` | Rotate −90° |
